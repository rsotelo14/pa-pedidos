Documentación del Proyecto de Procesamiento de Pedidos

Descripción

Este proyecto de procesamiento de pedidos es una aplicación Java que permite realizar tareas específicas en la gestión de pedidos. A continuación, se describen los pasos para compilar y ejecutar el proyecto utilizando Maven.

Requisitos

Para compilar y ejecutar este proyecto, es necesario tener Maven instalado. Si no se cuenta con Maven, se puede instalar mediante pip con el siguiente comando:

pip install maven

Instrucciones de Uso

Compilación

Para compilar el proyecto, utilice el siguiente comando en el directorio raíz:

mvn clean compile

Ejecución

Para ejecutar la clase principal, use el siguiente comando:

mvn exec:java -Dexec.mainClass="com.empresa.Main"

Tests

Para ejecutar los tests, utilice el siguiente comando:

mvn test

Estructura del Proyecto

A continuación, se describe la estructura de clases del proyecto:

  • main/java/com/empresa
    • Main.java: Clase principal que inicia la aplicación.
  • main/java/com/empresa/modelo
    • Pedido.java: Clase que representa un pedido en el sistema.
  • main/java/com/empresa/servicios
    • ProcesadorPedidos.java: Clase que gestiona el procesamiento de los pedidos.
    • ProcesamientoPago.java: Clase que maneja los pagos.
    • Empaquetado.java: Clase encargada del empaquetado de pedidos.
    • Envio.java: Clase que gestiona el envío de pedidos.
    • PrioritizedTask.java: Clase auxiliar para tareas priorizadas.
  • main/java/com/empresa/utilidades
    • UtilidadesConcurrencia.java: Clase con utilidades para manejar concurrencia.
  • test/java/com/empresa
    • MainTest.java: Test para la clase principal.
  • test/java/com/empresa/servicios
    • ProcesamientoPagoTest.java: Test para la clase ProcesamientoPago.
    • EmpaquetadoTest.java: Test para la clase Empaquetado.
    • EnvioTest.java: Test para la clase Envio.
